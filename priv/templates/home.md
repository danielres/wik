# Home

## Welcome to your new wiki

### Creating pages

1. Start by creating links with `[[Page Name]]`
2. Then click on the link to create the page

Tip: pages can have children: `[[Recipes/Cakes]]`

### Inserting special content

* Use the slash menu (`/`) for special commands.

### Using tags

Insert `#tags` in headings to collect and find related info easily:

```
## Carrot cake #recipes
```

Click on the tag below to see it in action.
